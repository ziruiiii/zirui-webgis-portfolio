Community Gardening and Parks Access Map

This interactive ArcGIS JavaScript web mapping application helps users explore parks and green spaces in Singapore. Users can search for locations, toggle map layers, and draw a rectangle or polygon to analyse green-space access, park connector availability, national park presence, and the diversity of green-space categories within a selected area.

What this map can be used for

This application helps users explore the spatial distribution of parks and green spaces in Singapore. It can be used to identify whether a selected area has park and open space access, national park presence, and park connector access, while also summarising the diversity of green-space categories in the selected area.

Data used

The application uses an ArcGIS Online web map containing four green-space-related layers: Parks and Open Space, Park Connector, Park, and National Park. These layers are queried dynamically when the user draws a selection area on the map.

How to use

  1. Use the Search widget to locate a place in Singapore.
  2. Use the Layer List widget to turn layers on or off.
  3. Use the Sketch tool to draw a rectangle or polygon on the map.
  4. Read the result section above the map to interpret green space access and diversity.
  5. Click Clear Selection to remove the current analysis.
